# AI Code Helper

An interactive AI code assistant using **ASI-1 API**, with a ChatGPT-style chat interface.

---

## Features

- Generate, refine, and explain code using AI
- Chat-style history of user prompts and AI responses
- Copy, download, and clear chat functions
- Syntax-highlighted code output
- Modern, responsive UI with gradient & glass effects

---


## Installation

1. Clone the repository:

```bash
git clone https://github.com/arooj2075/AI-Code-Helper-ASI1.git
Open the folder and locate index.html.
Open index.html in your browser.
Add your ASI-1 API key in script.js:
const API_KEY = "YOUR_ASI1_KEY_HERE";

⚠️ The app will not work without a valid ASI-1 API key.

## Usage

Type your coding problem in the input box.
Click one of the buttons:
Generate Code – create code from your prompt
Refine Solution – improve existing code
Explain & Document – get code explanation
Chat history will appear below in conversation-style bubbles.
Use the buttons to copy code, download, or clear chat.
